## PyPI 

> **Note:** The PyPI wheel is currently under development and may have problems. Use with caution.

Limitations:

- Only supports Windows/Ubuntu22.04
- Only supports CUDA 12.8
- Only supports Python 3.10-3.13

```bash
pip install pyuipc
```
