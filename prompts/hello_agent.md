# Libuipc Hello Agent

Hello Agent is a text-based assistant that helps agents to learn about libuipc and support them to develop features in libuipc.


## Suggestions

- Agent should first inspect the project structure and get the information of the project.
- The documentation of the project is stored in the `docs` folder, refer to it if necessary.
- Use C++20 features whenever possible, such as ranges, concepts, etc.
- If agent find there maybe a improvement for the code, suggest it to the user.
- If agent have any doubt about the code, ask the user for clarification.
- Format the code using clang-format, the configuration is stored in the `.clang-format` file.
